## 1. 개요 (Overview)

본 프로젝트는 독자가 소설 속 캐릭터와 대화하며 작품을 더 깊이 이해할 수 독서 경험 플랫폼입니다.
주요 기능으로는 도서 탐색, 캐릭터와의 채팅, 시나리오 생성 등이 있습니다.

---

## 2. 인증 (Authentication)

### 2.1 로그인 (Login)

![Login](mocks/Login.png)

**설명**: 사용자가 이메일과 비밀번호를 입력하여 플랫폼에 접속하는 화면입니다.

**주요 기능**:
- **이메일 로그인**: 이메일 및 비밀번호 입력 필드. 유효성 검사 포함.
- **자동 로그인**: 'Remember me' 체크박스를 통한 세션 유지 기능.
- **언어 변경**: 우측 상단 언어 전환 버튼.
- **회원가입 이동**: 계정이 없는 사용자를 위한 회원가입 페이지 링크.
- **3D 배경**: 몰입감을 주는 3D 인터랙티브 배경.

**사용자 인터랙션**:
- 이메일/비밀번호 입력 후 'Log in' 버튼 클릭 시 인증 요청.
- 입력값 오류 시 붉은색 텍스트로 에러 메시지 표시.

### 2.2 회원가입 (Register)

![Register](mocks/Register.png)

**설명**: 신규 사용자가 계정을 생성하는 화면입니다.

**주요 기능**:
- **계정 정보 입력**: 이름, 이메일, 비밀번호, 비밀번호 확인 필드.
- **이용 약관 동의**: 서비스 이용 약관 체크박스.
- **유효성 검사**: 실시간 입력값 검증 (이메일 형식, 비밀번호 일치 여부 등).

**사용자 인터랙션**:
- 모든 필수 정보 입력 후 'Create Account' 버튼 활성화.
- 성공 시 로그인 화면으로 이동 또는 자동 로그인.

## 3. 메인 (Main)

### 3.1 홈 화면 (Home)

![Main](mocks/Main.png)

**설명**: 서비스의 랜딩 페이지로, 주요 기능으로의 빠른 접근과 추천 콘텐츠를 제공합니다.

**주요 기능**:
- **히어로 섹션**: 서비스 슬로건 및 소개 링크 ('What is Gaji?').
- **빠른 검색**: 도서 제목, 작가, 설명 등으로 검색할 수 있는 중앙 검색바.
- **인기 토픽**: 현재 트렌딩 중인 키워드나 주제를 보여주는 리스트 (토글 가능).
- **네비게이션**: 상단 헤더를 통한 주요 메뉴 이동 (Books, Conversations 등).

**사용자 인터랙션**:
- 검색바에 키워드 입력 후 Enter 시 검색 결과 페이지 이동.
- 인기 토픽 클릭 시 관련 도서 목록으로 이동.

### 3.2 소개 페이지 (About)

![About](mocks/About.png)

**설명**: 서비스의 미션, 비전, 사용 방법을 안내하는 페이지입니다.

**주요 기능**:
- **가이드**: 플랫폼 사용법에 대한 단계별 설명.
- **미션 스테이트먼트**: 서비스가 추구하는 가치 설명.
- **3D 비주얼**: 스크롤에 반응하는 3D 요소로 시각적 즐거움 제공 (Scrollytelling).

## 4. 도서 (Books)

### 4.1 도서 목록 (Book List)

![Books](mocks/Books.png)

**설명**: 등록된 모든 도서를 탐색하고 필터링할 수 있는 화면입니다.

**주요 기능**:
- **그리드 뷰**: 도서 커버와 제목을 카드 형태로 나열.
- **검색 및 필터**:
    - 텍스트 검색 (제목, 저자).
    - 장르별 필터 (Romance, Classic, Mystery 등) 버튼.
- **상태 표시**: 검색된 도서 수 표시.

**사용자 인터랙션**:
- 장르 버튼 클릭 시 해당 장르 도서만 필터링.
- 도서 카드 클릭 시 도서 상세 페이지로 이동.

### 4.2 도서 상세 (Book Detail)

![Book Detail](mocks/Book Detail.png)

**설명**: 특정 도서의 상세 정보와 등장인물 관계, 시나리오 생성 기능을 제공합니다.

**주요 기능**:
- **도서 정보**: 표지, 제목, 저자, 장르, 줄거리, 태그.
- **관계도 (Relationship Graph)**: 캐릭터 간의 관계를 시각화한 인터랙티브 그래프.
- **캐릭터 목록**: 대화 가능한 캐릭터 리스트. 선택 시 시나리오 모달 오픈.
- **좋아요/공유**: 도서에 대한 관심 표시 및 공유 기능.

**사용자 인터랙션**:
- 관계도 노드 클릭 시 해당 캐릭터 정보 하이라이트.
- 'Talk' 버튼 클릭 시 대화 시나리오 설정 모달 팝업.

### 4.3 통합 검색 (Integrated Search)

![Integrated Search](mocks/Integrated Search.png)

**설명**: 도서, 대화, 캐릭터 등 모든 리소스에 대한 검색 결과를 통합하여 보여줍니다.

**주요 기능**:
- **카테고리별 결과**: 도서, 대화, 캐릭터 섹션으로 나누어 결과 표시.
- **검색 필터**: 상세 조건을 통한 결과 좁히기.

## 5. 대화 (Conversations)

### 5.1 대화 목록 (Conversation List)

![Conversations](mocks/Conversations.png)

**설명**: 사용자가 참여하거나 저장한 대화 목록을 확인하는 페이지입니다.

**주요 기능**:
- **대화 내역**: 최근 대화 순으로 카드 리스트 표시.
- **필터링**: 진행 중인 대화, 완료된 대화 등 상태별 필터.
- **요약 보기**: 대화의 마지막 내용이나 요약 정보 미리보기.

**사용자 인터랙션**:
- 대화 카드 클릭 시 채팅 화면으로 이어하기.

### 5.2 캐릭터 채팅 (Character Chat)

![Character Chat](mocks/Character-chat-page.png)

**설명**: AI 캐릭터와 실시간으로 대화를 나누는 핵심 화면입니다.

**주요 기능**:
- **채팅 인터페이스**: 말풍선 형태의 대화창. 사용자(우측), 캐릭터(좌측).
- **입력창**: 텍스트 입력 및 전송.
- **분기 선택**: 스토리 진행에 따른 선택지 제공 (Fork Navigation).
- **메모장**: 대화 중 중요한 단서나 내용을 기록하는 기능.
- **대화 제어**: 대화 저장, 종료, 되감기 등.

**사용자 인터랙션**:
- 메시지 입력 후 전송 시 캐릭터의 실시간 응답 표시 (Typing Indicator).
- 선택지 발생 시 버튼 클릭으로 스토리 전개 방향 결정.

## 6. 사용자 (User)

### 6.1 프로필 (Profile)

![Profile](mocks/Profile.png)

**설명**: 사용자의 활동 내역과 개인 설정을 관리하는 화면입니다.

**주요 기능**:
- **프로필 정보**: 닉네임, 프로필 이미지, 팔로워/팔로잉 수.
- **활동 피드**: 내가 쓴 댓글, 좋아요한 도서, 진행 중인 대화 리스트.
- **설정**: 계정 설정 및 알림 설정 진입.

**사용자 인터랙션**:
- 탭 메뉴(Overview, Conversations, Likes 등)를 통해 섹션 전환.
